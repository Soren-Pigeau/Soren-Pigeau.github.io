# assets

Ressources statiques du portfolio (images, etc.).

## photo.png

Ma photo de profil, au format portrait (~800×1000 px).
Le code charge `assets/photo.png` automatiquement au chargement.
Sans fichier, une silhouette stylisée s'affiche en fallback.

Si tu utilises un autre format (`.jpg`, `.webp`), modifie le `src` dans `index.html` :

```html
<img src="assets/photo.png" alt="Soren Pigeau" ... />
```
